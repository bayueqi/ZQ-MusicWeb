# 🌟 概述

ZQ-MusicWeb 是一个简单的音乐搜索、下载和播放网站。
它提供了一个轻量级、浏览器友好的界面，可以直接从您的 GitHub Pages 站点搜索、播放和下载音乐。



## ✨ 主要功能

- 🎵 **在线音乐搜索与播放**  
  支持通过关键词搜索歌曲，并直接在浏览器中播放。

- 📻 **多个音乐来源**  
  集成了咪咕、网易、酷我和 QQ 等在线平台。

- 💛 **可爱简洁的界面**  
  以皮卡丘为主题的简洁、有趣的界面，适合轻松聆听音乐。
